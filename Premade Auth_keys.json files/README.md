# Pre made auth_keys.json
*None of the files in here will work in the FreeSoul ACN server, only custom servers you manually add these Auth_keys.json files to.

This folder has pre made auth_keys.json files incase you don't want to make them. 

Currently there is only 1 pre made auth_keys.json file, but more will be added if needed. 

# IMPORTANT INFO:

- **Change the keys!:** The keys in the pre made auth_keys.json files need to be changed, other wise anyone who knows of this repo can easily use any of the options you made. The keys are the 3 digit number at the top of each option, just edit that to what ever you would like.

- **Rename the pre-made file to "auth_keys.json":** The pre made auth_key.json files hosted are renamed to let you know what they currently have in them. The ACN will not notice the file, unless you rename it to auth_keys.json.

- **Place the Pre-made Auth_keys.json file you pick in your server folder:** The auth_keys.json file will need to be placed in the same folder as the server exe/server DLLs. If you already have one there, you can just replace it.
